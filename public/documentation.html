Documentação

Guia completo de uso e referência da plataforma NovaStocks

1. Visão Geral

A NovaStocks oferece ferramentas avançadas para análise de ações, notificações de eventos e geração de relatórios. Nesta documentação você encontrará guias de uso, referência de API e melhores práticas.

2. Guia de Início Rápido

  1. Cadastre-se em signup.html.
  2. Faça login em login.html.
  3. Acesse o painel e configure seus alertas de ações.
  4. Explore a API pública em /api para buscar cotações e indicadores.

3. Referência de API

3.1 Buscar Cotações

GET /api/stock?symbol={ticker}

Retorna o preço atual e indicadores básicos para o ticker informado.

3.2 Listar Páginas CMS

GET /api/webhook?cms=list

Retorna lista de slugs das páginas HTML publicadas via CMS.

3.3 Salvar Página CMS

POST /api/webhook?cms=save

Content-Type: application/json

{ "slug": "exemplo", "content": "
Olá
" }

4. Exemplos de Uso

Exemplo de fetch em JavaScript para buscar cotações:

fetch('/api/stock?symbol=NVDA')

.then(res => res.json()) .then(data => console.log(data));

5. Perguntas Frequentes

Posso usar a API gratuitamente?
Sim, o Plano Básico oferece chamadas de API ilimitadas para indicadores básicos.
Como configuro notificações?
No painel de usuário, vá em "Alertas" e defina eventos de data ou preço para cada ticker.
