Início Rápido

  1. Cadastre-se em signup.html.
  2. Faça login em login.html.
  3. Configure seus alertas em Alertas.
  4. Consuma nossa API: GET /api/stock?symbol=NVDA.

Referência de API

GET /api/stock?symbol={ticker}

Retorna preço e indicadores básicos.

GET /api/webhook?cms=list

Lista slugs de páginas publicadas via CMS.

POST /api/webhook?cms=save

{ "slug": "exemplo", "content": "
Olá
" }

Exemplos de Uso

fetch('/api/stock?symbol=AAPL')

.then(r => r.json()) .then(data => console.log(data));

Perguntas Frequentes

API é gratuita?
Sim, ilimitada no Plano Básico para indicadores essenciais.
Como ativar notificações?
No painel, acesse Alertas e configure eventos de data ou preço.
